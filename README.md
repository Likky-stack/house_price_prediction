# House Price Prediction using Linear Regression

## 🎯 Objective

To build a machine learning model that predicts house prices based on features such as area, number of rooms, and location using Linear Regression.

## 📌 Overview

This project uses Linear Regression to estimate house prices from historical housing data. It analyzes how different factors like size, bedrooms, and location influence property prices.

## 🛠 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## 📂 Dataset

The dataset contains housing-related features such as:

* Area (square feet)
* Number of bedrooms
* Number of bathrooms
* Location
* Price (target variable)

## ⚙️ Workflow

* Data preprocessing and cleaning
* Handling missing values
* Feature selection
* Splitting dataset (train & test)
* Model training using Linear Regression
* Model evaluation using metrics (R² score, Mean Squared Error)

## 📊 Results

* Successfully trained a Linear Regression model to predict house prices
* Evaluated model performance using R² score and error metrics
* Visualized relationship between features and house prices

## 📈 Key Insights

* House price increases with area and number of rooms
* Certain features have stronger influence on price prediction

## 🚀 Future Improvements

* Use advanced models (Random Forest, XGBoost)
* Improve accuracy with feature engineering
